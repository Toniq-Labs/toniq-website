Cross-Chain Bridge

atomIC

atomIC is a cross-chain atomic bridge protocol built on DFINITY's Internet Computer. atomIC allows for the transfer of digital assets using secure smart contracts/canisters and a stake-based consensus mechanism.


  • Threshold consensus
  • Ethereum bridge (ERC20 token support)
  • Binance Smart Chain bridge (BEP-20 token support)
  • Tezos bridge (FA token support)
Unlimited Exchange Platform

Exponent

We are currently building out our token platform Exponent. This will provide users with tools and on-chain services to design, deploy and manage NFT and traditional tokens. Exponent will also provide decentralized marketplaces.


  • EXT - our multi-token canister for NFT and stackable tokens
  • eDEX protocol - for trading through the Exponent Marketplace
  • Exponent SDK - tooling for developers
  • Rapt Interface - ICP tokens as an EXT
